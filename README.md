## 📖 详细用法

### 🌤 天气查询

**支持任意中国城市，说人话就行：**

```
北京明天天气怎么样？
上海这周末要下雨吗？
成都今天穿什么合适？
```

**返回示例：**

```
🌤 北京 · 明天（周二）

🌡 最高 12°C，最低 3°C
💨 北风3级，湿度 45%
🌥 晴转多云，降雨概率 8%

👔 穿衣建议：厚外套 + 毛衣
☂️ 不需要带伞
```

**支持时间范围：**
- `今天` / `明天` / `后天`
- `本周` / `这周末` / `未来7天`

---

### 💱 汇率换算

**自然语言输入，直接给结果：**

```
500美元能换多少人民币？
1000元人民币换日元是多少？
今天欧元对人民币汇率
```

**返回示例：**

```
💱 实时汇率 · 2024-03-09 23:00

1 USD = 7.2156 CNY

500 USD = ¥3,607.80

📈 近7日走势：人民币小幅升值 0.3%
💡 建议：当前汇率处于近期高位
```

**支持货币：**
USD · EUR · JPY · KRW · HKD · GBP · AUD · CAD · SGD · THB 等20+种

---

### 📰 新闻摘要

**按领域获取今日要闻，AI提炼关键信息：**

```
今日科技新闻
今天财经有什么大事
AI行业最新动态
```

**返回示例：**

```
📰 今日科技要闻 · 3条精选

1. 【OpenAI】发布GPT-5，推理能力大幅提升
   → 核心：比GPT-4o快3倍，API价格下降40%
   🔗 查看原文

2. 【华为】新款Mate 70系列开售，首日销量破百万
   → 核心：搭载麒麟9010，国产芯片占有率再创新高
   🔗 查看原文

3. 【比亚迪】宣布进军东南亚市场，泰国建厂投产
   → 核心：计划2025年产能达20万辆/年
   🔗 查看原文
```

**支持领域：**
科技 · 财经 · 股市 · AI · 房产 · 国际 · 体育 · 娱乐

---

### 📦 快递追踪

```

查快递：京东 JD1234567890123
我的包裹到哪了？（需提供单号）
```

**返回示例：**

```
📦 顺丰速运 · SF1234567890

✅ 最新状态：派件中

📍 物流轨迹：
  03-09 10:30 · 【北京朝阳区】快递员正在派送，电话：138****1234
  03-09 08:15 · 【北京顺义转运中心】已发出
  03-08 23:40 · 【上海浦东转运中心】已到达
  03-08 18:20 · 【发件方】已揽收

⏰ 预计今天18:00前送达
```

**支持快递公司：**
顺丰 · 圆通 · 中通 · 韵达 · 申通 · 百世 · 京东物流 · 邮政EMS · 菜鸟

## 🛠 技术说明

本工具通过以下方式获取数据：
- 天气：[wttr.in](https://wttr.in) + [Open-Meteo](https://open-meteo.com)
- 汇率：实时汇率公开数据源
- 新闻：主流中文新闻网站
- 快递：快递100等公开查询接口

**无需任何付费API，完全开源免费。**
